Gardenia（nailsalon-A）

## 概要
個人営業で施術人数が限られている自爪育成を目的とした、プライベートネイルサロン予約専用ランディングページ（LP）**です。
常連客の予約ページ、新規顧客の拡大を目的とし、30代後半〜50代をターゲットに予約への導線設計とブランドの世界観を両立させたデザイン・実装を行いました。

---

コーディング：12.5時間

## 実装内容
- セクション構成: コンセプト /　問題定義　/　共感　/ こだわり　/　フロー　/　 料金案内 /  店舗情報 / CTA（予約ボタン）
- レスポンシブ設計: スマートフォンでの閲覧・予約操作をメインに想定したモバイルファースト設計
- インタラクション:
- 直感的に操作できる予約フォームへの誘導ボタン

## 制作意図（課題解決アプローチ）
- 1. ターゲットと目的の明確化
ターゲット: 30代後半〜50代。自分を労わりたい大人世代の女性。看護師、教員、事務職など「派手なネイルはできないが、手元を綺麗に保ちたい」専門職の方がメインターゲット。
課題: 電話予約がメインのため作業効率化したい。また新規顧客拡充。
解決策: 空き情報の確認ができるCTAを配置し作業効率を上げるとともに、従来取りこぼしていた顧客の回収を行う。
xxx

- 2. デザインのこだわり
女性雑誌のような（エディトリアルな）タイポグラフィを採用。
ナチュラルな配色で落ち着いた女性らしさをイメージ

## 工夫した点
- CTA（行動喚起）の配置: ユーザーがスクロール中にいつでも予約を検討できるよう、視認性の高いCTAボタンを適切なタイミングで配置。
- 表示速度の意識: 画像の最適化を行い、モバイル環境でもストレスなく閲覧できるよう配慮。
