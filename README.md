Gardenia（nailsalon-A）

## 概要
派手なネイルはできないが、手元を美しく保ちたい」専門職の女性をターゲットとした、自爪育成特化型プライベートサロンの予約専用LPです。

## 制作意図（課題解決アプローチ）
- 1. 専門職のターゲットに刺さる「共感型」ストーリー
 - ターゲット: 看護師、教員、事務職など、職業柄派手な装飾はできないが、乾燥や形などの爪の悩みを解消したい30代後半〜50代の女性。
 - 構成: 「爪を隠す悩み」に寄り添う問題定義から始まり、自爪育成の専門性を説くことで、ターゲットが抱く「ここなら私の悩みを解決してくれる」という期待感を醸成しました。
- 2. 信頼を醸成するエディトリアル・デザイン
 - 視覚戦略: 落ち着いた大人世代に響くよう、ベージュやグレージュを基調としたナチュラルな配色を採用。雑誌のような洗練されたタイポグラフィと広めの余白により、サロンの上質さと清潔感を表現しました。
 - 機能美: 装飾過多を避け、情報を整理することで、迷わず予約（CTA）へ辿り着けるユーザビリティを確保しています。
- 3. 予約導線の最適化による実務改善
 - 解決策: これまでメインだった電話予約をオンラインへ誘導することで、施術中の手を止めることなく予約を受け付けられる体制を構築。機会損失を防ぎ、店舗運営の効率化を支援する設計にしました。

## 工夫した点
- モバイルファーストのUI設計: 仕事の合間や移動中に閲覧・操作するユーザーを想定し、片手で操作しやすいボタン配置と、ストレスのないスクロール体験を追求。
- 表示パフォーマンス: 画像の最適化を行い、サロンの高品質なイメージを損なわずに高速なページ読み込みを実現しました。
